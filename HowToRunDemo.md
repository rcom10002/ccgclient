# 注意事项 #

不要轻易告诉他人游戏后台管理中心的用户名与密码

启动游戏时，请将超链接地址中的“**IP地址**”替换成你的测试服务器的IP地址，如

```
假设服务器IP地址为192.168.1.1，那么要将以下地址

http://IP地址:8080/f3s/content/ccga/Administration.html

http://IP地址:8080/f3s/content/ccgc/CCGameClient.html

替换成

http://192.168.1.1:8080/f3s/content/ccga/Administration.html

http://192.168.1.1:8080/f3s/content/ccgc/CCGameClient.html
```

# 访问游戏后台管理中心 #

http://IP地址:8080/f3s/content/ccga/Administration.html

进入管理中心后，可以通过左侧菜单“でく・ロボット”打开监控控制台，最下面的按钮用于启动电脑，其中数字选项代表要启动的电脑个数，默认启动四个电脑。

  * 启动后，可以观摩四个电脑互相对战。
  * 如果要亲自参与游戏，可以只启动三个电脑，然后通过下节“**访问游戏前台客户端**”来加入游戏。
  * 如果需要两人参与，则只启动两个电脑，，然后通过下节“**访问游戏前台客户端**”来加入游戏，以此类推。

# 访问游戏前台客户端 #

http://IP地址:8080/f3s/content/ccgc/CCGameClient.html

如果是通过后台管理中心启动的电脑，请不要人工干预操作。只有正常通过游戏客户端登录的，才可以进行人工操作。
